git clone:
Создает локальную копию удаленного репозитория, включая всю его историю коммитов и файлы.
git status:
Показывает состояние репозитория, включая измененные, добавленные в индекс и новые файлы.
git add:
Добавляет изменения из рабочего каталога в «индекс» (staging area), подготавливая их к коммиту.
git commit:
Фиксирует изменения из индекса в локальном репозитории, создавая «снимок» состояния проекта с комментарием.
git push:
Отправляет ваши локальные коммиты на удаленный репозиторий.
